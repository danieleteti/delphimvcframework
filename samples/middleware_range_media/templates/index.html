Setup — Adding Sample Media Files
Download sample files with get_samples.bat

Run get_samples.bat (Windows) or get_samples.sh (Linux/macOS) in the project folder. The scripts download sample.ogg (Beethoven — Moonlight Sonata, Public Domain) and sample.mp4 (Big Buck Bunny, CC BY 3.0 © Blender Foundation) into the media/ folder using curl.

Build and run the project, then open http://localhost:8080.

Alternatively, drop any compatible media file into the media/ folder and serve it directly at /media/<filename> — no server restart needed.

Test with curl:

# Full file (200 OK)
curl -I http://localhost:8080/media/sample.ogg

# First 64 KB (206 Partial Content)
curl -v -H "Range: bytes=0-65535" http://localhost:8080/media/sample.ogg

# Last 1 KB (206 Suffix Range)
curl -v -H "Range: bytes=-1024" http://localhost:8080/media/sample.ogg

# Invalid range (416 Range Not Satisfiable)
curl -v -H "Range: bytes=9999999-" http://localhost:8080/media/sample.ogg
